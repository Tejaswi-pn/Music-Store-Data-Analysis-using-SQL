# Music-Store-Data-Analysis-using-SQL

## 📌 Project Overview

This project analyzes a Music Store database using MySQL. The objective is to solve real-world business problems by writing SQL queries that extract meaningful insights from customer purchases, artists, albums, invoices, and sales data.

The project demonstrates SQL skills including:

- Database Design
- Data Import
- Joins
- Aggregate Functions
- Group By
- Order By
- Window Functions
- Common Table Expressions (CTEs)
- Subqueries
- Business Problem Solving

---

## 📂 Dataset

The project uses multiple CSV files representing a music store database.

Tables include:
